# Data Files

Expected workflow:

1. `data/raw/hackernews_comments.csv`
   - Generated by `uv run takemeter-collect-hn`.
   - Contains public HN comments with empty labels.

2. `data/labeled/takemeter_hn_review_queue.csv`
   - Generated by `uv run python -m ai201_project3_takemeter.heuristic_prelabler`.
   - Contains rough labels and must be manually reviewed.

3. `data/labeled/takemeter_hn_labeled.csv`
   - Final reviewed dataset for Colab upload.
   - Must contain at least 200 rows.
   - Required columns: `text`, `label`.
   - Useful extra columns: `source`, `story_title`, `notes`, `review_status`.

Validate the final dataset:

```bash
uv run takemeter-validate-dataset data/labeled/takemeter_hn_labeled.csv
```

For the rough review queue only, add `--allow-unreviewed` so the command checks
count and distribution without pretending the labels are final.
